# Trivia Quiz (TCP Client/Server)

## Run
1. Start server: `python TriviaQuizServer.py`
2. Start client: `python TriviaQuizClient.py`

## Play
- Choose a category and answer A/B/C/D.
- Points: Easy 10, Medium 20, Hard 30.
- Menu options: Start Quiz, Leaderboard, My Stats, Quit.

## Configure
- Server IP: edit `serverName` in TriviaQuizClient.py
- Port: edit `serverPort` in both client and server

## Notes
- Data is in-memory; scores reset when server restarts.
